‹Id› → _‹IdRem›

Алфавиты (VT и VN)

Терминальные символы (VT): const, a, b, c, ..., z, A, B, C, ..., Z, 0, 1, 2, ..., 9, _, :, &, str, =, ", ;, !, @, #, $, %, ^, *, (, ), -, +, {, }, [, ], ', <, >, ,, ., ?, /, \, |, ~, \``, "`.

Нетерминальные символы (VN): ‹Const›, ‹Id›, ‹IdRem›, ‹LinkType›, ‹StrType›, ‹Equal›, ‹String›, ‹StringRem›, ‹EndOperator›, ‹Digit›, ‹Letter›, ‹Symbol›.
