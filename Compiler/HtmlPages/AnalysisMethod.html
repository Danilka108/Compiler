Метод анализа

Для разбора грамматики был выбран метод рекурсивного спуска.

Метод рекурсивного спуска основывается на том, что каждой нетерминальной конструкции грамматики соответствует конкретная часть кода, т.е. процедура или функция. Например, в языке С можно использовать функцию, которая распознает последовательность символов, соответствующую данному нетерминалу. Эти функции или процедуры вызываются в порядке, определенном правилами грамматики, они могут вызывать сами себя рекурсивно.

Таким образом, для использования этого метода необходим язык программирования, который поддерживает рекурсию. Реализация метода рекурсивного спуска для анализа синтаксиса строковых констант в языке Rust представлена в приложении В.
