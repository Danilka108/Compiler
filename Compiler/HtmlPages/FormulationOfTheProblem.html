Постановка задачи

Константы – это элементы данных, значения которых известны и в процессе выполнения программы не изменяются.

Для описания констант в языке Rust используется служебное слово const.

Формат записи: const имя_константы: &str = “значение”;

Примеры

  • Строковая константа: const constant_string: &str = “constant string value”;
  • Строковая константа с экранированием: const constant_string: &str = “constant string \”value\””;
