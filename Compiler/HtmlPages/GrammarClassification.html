Классификация грамматики

Согласно классификации Хомского, грамматика G[‹Const›] является автоматной. Правила (1)-(14) относятся к классу праворекурсивных продукций (A → aB | a | ε): 

‹Const› → const
‹Id› → _‹IdRem›
‹Id› → ‹Letter›‹IdRem›
‹IdRem› → ‹Letter›‹IdRem›
‹IdRem› → ‹Digit›‹IdRem›
‹IdRem› → _‹IdRem›
‹IdRem› → :‹LinkType›
‹LinkType› → &‹StrType›
‹StrType› → str‹Equal›
‹Equal› → =‹String›
‹String› → “‹StringRem›;
‹StringRem› → \”‹StringRem›
‹StringRem› → ‹Symbol›‹StringRem›
‹StringRem› → ”‹EndOperator›

Отметим, что правила должны быть либо только леворекурсивными, либо только праворекурсивными. Комбинация тех и других не допускается. Данная грамматика содержит только праворекурсивные продукции (1)-(14), и, следовательно, является полностью автоматной.
